Update System Software

Important note, linux does not need to reboot unless the kernal needs to updated

Anything borded in red means it doesn't look like what he did in the tutoiral, but seems correct

 The dvd install might be months old so it's important to
update your software.  Networking needs to be turned on to do any updating.
Click the icon in the top right corner, under Wired off
select connect

 Then go to Applications→System Tools→Software

 Click on the Software tab and select Check for Update
He mentions hitting the refresh tab because this isn't always correct.

Use command line for updates (CentOS 7 dnf)

 It's better to use the command line when checking for updates because
it is more accurate and Up to Date. Go to Applications→System tools
and drag the Terminal application to your desktop and open it.

 Type in:

sudo dnf update

if the command is not found type if running CentOS 7

sudo yum makecache sudo yum -y install dnf

 Hit yes for all options for dnf update,
Once done, reboot by clicking the power button in the top right
corner of the window
