  • Guest OS has to be up to date
  • After update the OS has to rebooted to use
    latest kernal from OS Update
    (Restoring to previous snap shot does not accomplish this)
  • Networking needs to be turned on to install software
  • The user needs to be the Administrator

1.

 To update guest additions open the termainl and enter

sudo dnf update -y

 The dnf command handles software installations from the repository it has replaced the yum command.
The -y option answers all questions to yes

2.

 When done type in:

sudo reboot

And hit enter

3.

 After reboot, log back and ensure the network is on

4.

 Open the terminal and type the command:

sudo dnf group install -y "Development tools"

5.

 After this package is installed, install one more by typing:

sudo dnf install -y elfutils-libelf-devel

6.

 After this you have to insert the virtual guess edition cd.
To do this go to the Devices tab
This will take a few minutes because it has to compile a new
kernal module

 You can now change the display resolution
