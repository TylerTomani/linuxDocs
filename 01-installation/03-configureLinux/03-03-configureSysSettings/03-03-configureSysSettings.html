03-03-Configure basic system settings

 After guest installations are complete, now would be a good
time to,

  • set a host name
  • set date and time
  • change language
  • edit shortcuts

1.

 Set the network to auto connect, Go to the main Settings→Network

2.

 Under Wired make sure connection is turned on

3.

 To set up auto connectclick the gear icon
and select auto automatically, Then click Apply

4.

 To change the host name click on the details tab at the bottom left

5.

 There you can change the host name

6.

 You can have time changed automatically
or set it up manually

 There is one lower level configuration setting you might
want to change at this time.

 Under the GUI there was a display server that manages

  • Video card drivers
  • and how the Graphical Interface is drawn

 In the past this was X.org, but now this configuration is found in Wayland, and Enterprise Linux 8 uses Wayland by default

7.

 Open the terminal and type in the command

gedit admin:///etc/gdm/custom.conf

 If it doesn't work, type in

sudo

 This is the file that configures the dispay server,
After typing the command, enter your password, this will Open
up the file, uncomment the line with WaylandEnable=false
 Click save, then reboot, after reboot create a new snap shot
