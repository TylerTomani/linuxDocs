03-04-Customize GNOME 3 desktop

Really IMPORTANT !!

 Make the VM run faster by turning off animation in Tweaks

1.

 Use GNOME Tweaks porgram to modify GNOME desktop.

 It is located under,
Accessories→Tweaks

2.

 This is where you can modify the appearance of the desktop,

3.

 To download more options, go to,

gnome-look.org

 Choose the Prog-Gnome-theme as, b/c the Grant McWilliams chose that one

4.

 To extract it, Right click it, choose,
Open With Archive Manager
Then choose the location.
I put it back into the original location

5.

 Now copy the extracted file,

6.

 Go to the home directory

7.

 Select the hamburger icon and create a folder in your home directory named

.themes

8.

 Click the hamburger icon again and select Show hidden files

9.

 open .themes and paste the extracted file in it

 To use the theme, close then reopen tweaks as seen at the top of the page
Then change it under this tab.

select Window and go to the bottom right
