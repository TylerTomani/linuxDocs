Virtual Machine snap shots

When running a new vm, take a snap shot using
cmd + t shortcut
This will save the original settings

Snapshots are usfule, if the system get infected with malware
You can always revert back to an early setting. Take
snap shots whenever you make changed you don't want to repeat

To revert back to previous snap shot, hit the x to close the window,
This will bring up a window for powering off, but instead choose the
Restore current snapshot
