Virtual Machine snap shots

 When running a new vm, take a snap shot using
cmd + T shortcut
This will save the original settings

 Snapshots are useful. Take snap shots whenever you
make changes you don't want to repeat.
For instance,

  • If you get a webserver up and running, take a snapshot
  • Ff the system get infected with malware, snapshot

You can always revert back to an early setting with a snapshot.

 To revert back to the most recent previous snap shot,
hit the x in the top left corner to close the window,
This will bring up a window for powering off, choose the
Restore current snapshot and Power off the machine

 Then click restart the vm by clicking start. This will restore the vm back to
the snapshot

 To revert back to any other snap, you must do it from the
VirtualBox Manager Window, click on the icon next to the Vm name
and click Snapshots

 Here you can take a snapshot with the camera icon,
this is the same as hitting host (cmd) + t

 You can not revert to another snapshot while the vm is running,
close the vm by hitting the x in the top left corner, select
Power off the machine but uncheck Restore current snapshot

 Go back the Vm manager snapshots window, choose the snap shot
and click restore
