Install from DVD

  To install from the DVD, we need to download an ISO image.
go to https://www.centos.org/download/
and select x86_64

 Ideally you want to choose a mirror close to you, but
you can select any if you don't know where they are.

 Click the CentOS-7-x86_64-DVD-2009.iso link to download.

 Once iso image has downloaded go to Virtual Box and your VM, then
click the Start Icon

Error

If you are running Virtual Box for the first time, you will probably encounter
this error, go to your Mac System Prefrences → Security and Privacy
an allow Oracle access. You will have to restart after.

 After restarting go back to your VM in Virtual Box, hit start again.
This time is will say,

 Go back to Sys Pref Security and Privacy, click on the Privacy tab
scroll down to the Input Monitoring and check VirtualBox.

 Now, go back to your VM in Virtual Box, and select start. Once in start,
click the folder icon.

 Then click the Add Icon.

 Navigate to where you downloaded the iso and select it

 After clicking on the downloaded iso, click choose, then start.

Inside VM, host key

  THE MOUSE might NOT WORK. Hit the host key "command" to use it.
 A window with black background will appear displaying:

  • Install CentOS Linux 8
  • Test this media & ..

The Test this media ... option is for download from physical disc,
NOT iso mirrors. SO, Choose Install CentOS Linux 8

 After Install is complete, this page should show up,
If mouse isn't working properly, place pointer in all four
corners of the virtual box window. If you can't get the mouse
to work just close/Power Off Machine and hit start again.
Once working and installed Click on continue

Operating System Install

 Next the installation summary screen will appear,
This is where you configure how the operting system will
be installed.

 First, choose time and date,
You can also set up language and keyboard

 Click on software selection and click on workstation,
The tutorial says workstation but only
Development and Creative Workstation is an option
So choose that and see what happens. Hit done

 Now choose the Installation Destination. Choose
ATA VBOX HARDDISK,
Under Storage configuration, Automatic
sets up 3 partitions, Custom, Allows us to the partitions we've created, choose Custom

 After clicking on "I will configure partitions", just click on
click to create them automaticall because my mouse keeps
dissapearing and I'm constantly restarting. Hit done go back to menu

 go to menu, once in menu, Hit begin installation.
A new menu showing root password and user creation will appear.
 NEVER LOG in as ROOT on linux, Always Create a USER.
First,
set up password for Root. NEVER LOG in AS ROOT, Then
create user.
Besure to hit check box in user creation to set up user as
administator.

Begin Installation, after rebooting, click top option in black screen,
agree to licence, log in as user, choose default option for keyboard

Everything should be installed, go back and take more screen shots, if you're confused, Do not forget to set up user and administator
