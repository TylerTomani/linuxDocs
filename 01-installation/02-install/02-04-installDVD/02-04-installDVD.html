Install from DVD

  To install from the DVD, we need to download an ISO image.
go to https://www.centos.org/download/
and select x86_64

 Ideally you want to choose a mirror close to you, but
you can select any if you don't know where they are.

 Click the CentOS-7-x86_64-DVD-2009.iso link to download.

 Once iso image has downloaded go to Virtual Box and your VM, then
click the Start Icon

Error

If you are running Virtual Box for the first time, you will probably encounter
this error, go to your Mac System Prefrences → Security and Privacy
an allow Oracle access. You will have to restart after.
