Creating a Virtual Machine

click here to get to first Problem #1

1.

 Before you install a virtual machine you must create one.
To do so, open the Virtual Box GUI and click on New

2.

 The first time I installed it, I did
Type: Mac OS X
Version: Max OS X (64-bit)

 The Second time I installed it, I did
Type: Linux
Version: Oracle (64-bit)
Because it came up by default

Should I do this??
Type: Linux
Version: Max OS X (64-bit)
or
Type: Linux
Version: Red Hat (64-bit)

3.

Things run smoother with at least 1.5 GB of memory, so choose 1500 mb

4.

Create a virtual hardrive. Choose the Create a virtual hard drive now option

5.

Keep the default VDI (VirtualBox Disk Image) selected

6.

 Choosing Fixed size will allocate full memory on host, even if it is not being used.
For that reason keep Dynamically allocated selected

7.

 Now choose the size of the drive. For starting, change the size to 10GB

8.

  Now there is a new virtual machine running and your host OS is ready for installation
