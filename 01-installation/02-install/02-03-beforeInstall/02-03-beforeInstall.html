Before Installing

There are two ways to download linux

  • Full DVD installation
  • Network installation

Either way, you will have to download an ISO images from
a website

Full DVD

Is much larger but contains everything needed to install (9.3 GB),
and you will immediately need to update the OS after

Network Installation

Is much smaller, (700 MB), but updates the lates package so you
do not need to update the OS right away
