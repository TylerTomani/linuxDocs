Optimizing you Virtual Machine

Before installing, optimize the vm. Make sure the VM is highlighted and click on
the Setting icon

System

Click on the System tab , and ensure the Enable I/O APIC option
is checked

 Then choose the Processor tab. Choose 2 Processor(s), one
for the host and one for the VM. If you have more than two, you can
choose more. If you don't, the bottom will show a text displaying
Invalid settings detected

 Make sure Enable PAE/NX is checked. This allows 32bit
machines to access more than 4GB's of RAM

Go to the Acceleration tab. If on MAC, keep default
Hyper-V is best for Windows.
KVM is best for Linux

Make sure Enable Nested Paging is checked

Display

 Now click on display and increase Video Memory
to the max, and check Enable 3D Acceleration.
let Virtual Box choose the Graphics Controller
  The VM is now ready for installation

Storage

 Select the storage icon to see the drives. To add another drive,
click on the Controller: SATA icon and then the plus icon
on the bottom right

 You can also add hard drive controllers by clicking the left plus icon

 To access the VM from another computer, go to the network icon
and change the Attached to Setting to Bridged Adapter. NAT will allow your VM to see other computers on the network, but
others can access this VM without Bridged Adapater.
