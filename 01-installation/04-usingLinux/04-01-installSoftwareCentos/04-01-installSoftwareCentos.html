04-01-Installing common software in CentOS 8 (really 7 here)

 There are two ways of installing software from software repositories

  • The gui
  • The terminal

The GUI

1.

 Go to Software under Applications

2.

 Listed will be the software, and these are not the only softwares
that you can install, only the recommended

 You must double click them, to open them. There are roughly 7000.
Only a few are available through the GUI, for the rest you must use the
command line

The Terminal

3.

 In the command line type,

sudo dnf install -y gimp

4.

 Once completed, type in gimp

5.

 Or select GNU Image Manipulation Program under Graphic

6.

 Once gimp opens, select Single window mode

Install Libra - a forklift of oracle open office

7.

 For Libra, install a group of softwares because this saves time
To do type in

sudo dnf group install -y "Office suite and productivity"

 To remove this or any software change install with remove

sudo dnf group remove -y "Office suite and productivity"

8.

 They should now appear under Applications in
Office and Graphics

9.

 To find more avalailable software type,

sudo dnf list available

 This will be a long list

10.

 To see which softwares are installed type

sudo dnf list installed
